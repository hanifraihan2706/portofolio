Legal & Admin Automation

Auction Minutes Automation System

Problem
Manual preparation of auction minutes required repetitive reformatting, legal data entry, and multi-page layout adjustments — consuming up to 6 hours per document with high risk of human error.
Solution
Built a Python desktop application using python-docx and a Tkinter GUI to parse structured input and generate standardized, print-ready legal documents — complete with conditional per-page footers and multi-line address handling.

Supply Chain & Operations

Retail Stock Reconciliation Dashboard

Problem
Retail stores managing thousands of SKUs across display areas and storage had no efficient way to detect discrepancies between physical counts and system records — leading to slow, error-prone audits.
Solution
Developed an interactive dashboard that compares physical stock input against system data in real-time, auto-detects status (Match / Over / Short), and generates completion percentage metrics per product category.
📈
Dynamic SKU tracking across all areas
Faster audit cycles · reduced reconciliation errors
Google Sheets Data Processing Dashboard Design Inventory Reconciliation

AI-Powered Integration

AI-Powered Finance Tracker

Problem
Traditional expense tracking requires manual app entry or spreadsheet logging, creating enough friction that most users abandon consistent tracking within weeks.
Solution
Built a Telegram Bot integrated with the Gemini AI API and Google Apps Script. Users send natural language messages like "lunch 45k" — the system interprets, categorizes, and logs the transaction automatically to a structured spreadsheet.
💬
Zero-friction expense logging via chat
Natural language input · auto-categorization · cloud sync

Mobile Inventory Checking App
