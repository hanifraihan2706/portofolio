Auction Minutes Automation System

Supply Chain & Operations

Retail Stock Reconciliation Dashboard

Problem
Retail stores managing thousands of SKUs across display areas and storage had no efficient way to detect discrepancies between physical counts and system records — leading to slow, error-prone audits.
Solution
Developed an interactive dashboard that compares physical stock input against system data in real-time, auto-detects status (Match / Over / Short), and generates completion percentage metrics per product category.
📈
Dynamic SKU tracking across all areas
Faster audit cycles · reduced reconciliation errors

AI-Powered Finance Tracker

Problem

Solution

Web App · Google Apps Script

Digital Product Catalog Web App

Problem
Product information was scattered across spreadsheets with no accessible, shareable interface for internal teams or customers — making it difficult to browse, filter, and reference product data in real-time.
Solution
Built a web-based product catalog using Google Apps Script and Tailwind CSS, deployed directly from Google Sheets as the data source. The app renders product listings dynamically with a clean, SaaS-style interface accessible via a shareable URL — no server required.
🌐
Spreadsheet → live web catalog, zero hosting cost
Google Sheets as backend · instant deploy · shareable link
